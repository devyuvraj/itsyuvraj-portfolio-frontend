# itsyuvraj.com – Personal Portfolio

This is my personal resume portfolio built using Angular 20 with standalone component architecture. The site is deployed at [https://itsyuvraj.com](https://itsyuvraj.com) (coming soon).

## 🚀 Tech Stack

- Angular v20 (Standalone Components)
- SCSS
- Angular Routing
- Vercel / Firebase Hosting (deployment)
- Optional: NestJS + OpenAI API for chatbot integration

## 📁 Pages

- `/home` – Introduction, skills, resume
- `/about` – Summary, education
- `/contact` – Contact form
- `/chatbot` – AI Chatbot (WIP)

npm install
