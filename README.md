## 🚀 Key Features & Architecture

This repository isn't just a static template; it's architected to demonstrate scalable frontend patterns:

- **100% Centralized Content (`DataService`)**: All text, skills, experience, and projects are driven by a single headless-CMS-like Angular service (`src/app/services/data.service.ts`). The UI templates contain zero hardcoded data.
- **Integrated AI Assistant**: Features a custom-built floating chat widget integrated with a serverless backend connecting to LLMs, trained specifically on my professional background.
- **Modern Angular 17 Control Flow**: Built using the latest `@for` and `@if` template syntaxes for enhanced rendering performance.
- **Scroll-Spy & Intersection Observers**: Custom native implementations to handle smooth scroll-spy navigation and scroll-triggered reveal animations without relying on heavy external libraries.
- **Automated Tracking & SEO**: Fully integrated with Schema.org JSON-LD, Google Tag Manager, MS Clarity, and Open Graph tags for enterprise-grade analytics tracking.

## 💻 Tech Stack

- **Framework**: Angular 17 (Standalone Components & SSR)
- **Language**: TypeScript 5.4
- **Styling**: SCSS (Custom Design System, CSS Variables, Flexbox/Grid)
- **Backend/Forms**: Web3Forms API
- **AI Integration**: Serverless API / LLM capabilities

## 🛠️ Local Development

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/devyuvraj/itsyuvraj-portfolio-frontend.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   ng serve
   ```
   Navigate to `http://localhost:4200/`. The app will automatically reload if you change any source files.
